# HackXlerate Cyberverse 🚀

HackXlerate Cyberverse is a **cybersecurity-focused web application** built to demonstrate secure communication, SMS model integration, chatbot interaction, and interactive public assets.  
This repository contains both backend and frontend code, designed to accelerate learning and experimentation in cyber defense and secure app development.

---

## 📂 Project Structure
- **`model/sms/`** → Contains SMS-related models and scripts (Python-based).
- **`public/`** → Static assets (HTML, CSS, images, client-side scripts).
  - `chat.html` → Chatbot interface for real-time interaction.
- **`server.js`** → Node.js backend server entry point (Express + Socket.io).
- **`package.json` & `package-lock.json`** → Project dependencies and configuration.
- **`.gitignore`** → Ignored files for clean version control.

---

## ⚙️ Tech Stack
- **Backend:** Node.js (Express + Socket.io server)
- **Frontend:** HTML, CSS, JavaScript
- **Chatbot:** WebSocket-based real-time messaging
- **Modeling:** Python (SMS model integration)
- **Version Control:** Git & GitHub

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/vanshguptaa654-maker/hackxleratecyberverse.git
cd hackxleratecyberverse
```
### 2. Install dependencies
```bash
npm install
```
### 3. Run the server
```bash
node server.js
```
